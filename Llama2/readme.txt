This directory contains code to perform Zero Shot Learning on Llama2-7b model
Llama2-7b response is stored in llama2_response.txt

Llama2.ipynb - Notebook to train the Llama2 model on the Automotive dataset
llama2_response.txt - Model Response of the Llama2 model
flan-response-dataset-new.txt - Held out dataset used to Fine Tune the Llama2 model
Llama2FT_local - Notebook that contains code for Fine Tuning the model run on local GPU

Note: The respect notebooks run on local gpu have been added 'filename_local.ipynb'. If the Collaboratory link doesn't
work please refer to the locally run notebooks
